# Bike Buyers Analysis Dashboard

## Overview
This project analyzes a dataset of bike buyers to understand trends in purchasing behavior, demographics, and price sensitivity. It includes:

- **Raw Dataset**: The original Excel file with all the data.
- **Analysis Workbook**: Excel file with pivot tables, calculations, and insights.
- **Dashboard**: Visual representation of key metrics using charts and tables.
- **Screenshots**: Images of the dashboards and charts for quick reference.

---

## Files in this Repository

| Folder/File | Description |
|-------------|-------------|
| `dataset/bike_buyers_data.xlsx` | Original dataset |
| `analysis/bike_buyers_analysis.xlsx` | Pivot tables |
| `images/` | Screenshots of dashboards and charts |
| `README.md` | Project overview and instructions |

---

## Key Insights
- Average sales and purchasing patterns by region and customer type.
- Demographic trends affecting bike purchases.
- Visualization of price sensitivity and product preference.

---

## How to Use
1. Open `bike_buyers_analysis.xlsx` in Excel to explore pivot tables and calculations.
2. Refer to the `images/` folder to view the dashboards visually.
3. Use the raw dataset to perform your own analyses or extend the project.
